Вот наиболее часто используемые методы (функции) из библиотеки **Telegram.Bot** версии **22.9.0** (поддерживает Bot API 9.4). В версии 22.x убрали суффикс `Async` из названий методов — все публичные методы асинхронные по умолчанию.

Список составлен на основе реальной практики, примеров из репозитория TelegramBots/Telegram.Bot.Examples, документации и популярных шаблонов использования в 2025–2026 годах.

### Краткие заметки по самым популярным комбинациям (2026)

- **Базовый запуск бота**  
  `GetMe` → проверка токена  
  `StartReceiving(updateHandler, errorHandler, receiverOptions)` → основной polling-цикл

- **Типичный обработчик сообщений**  
  `if (update.Message?.Text != null)` → `SendMessage` или `EditMessageText`

- **Inline-кнопки**  
  Получение `update.CallbackQuery` → **обязательно** `AnswerCallbackQuery` (иначе крутилка не исчезнет) → `EditMessageText` / `SendMessage`

- **Самые частые ReplyMarkup**  
  `ReplyKeyboardMarkup` (обычная клавиатура)  
  `InlineKeyboardMarkup` (inline-кнопки)  
  `ReplyKeyboardRemove` (убрать клавиатуру)
